# Notepad MERN app 


## Installation: 
- Install docker. 
- Install docker-compose. 

## Get started :
- clone the repo OR download only the `docker-compose.yml` file. 
- Run `docker-compose up` to open the application.

View Client on localhost:3000
View api    on localhost:9000
